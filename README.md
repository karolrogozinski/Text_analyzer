# Text_analyzer
text analysis app based on paradigms

## Features:
 - Open file
 - Add filters (with union or intersection)
 - Remove filters
 - Run Analyse 

## Make it possible to find word pattern:
 - Longer than
 - Shorter than
 - Equal length
 - Starts with sequence
 - Ends on sequence
 - Contain sequence
 - Custom, based on regex
